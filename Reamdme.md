# ScrapydDashboard
## 爬蟲儀表板
- crawler
  - Scrapy爬蟲服務
- reverse_proxy_server
  - Nginx反向代理伺服器
- schedule
  - 執行爬蟲工作的排程服務
- web_site
  - 爬蟲儀表板網站     
## 執行方式
``` bash
docker-compose up -d --build
```
